# EGG-Project

Repo gốc chứa 4 phần tách biệt, chỉ chung thư mục gốc:

- EGG-Chain: Rust workspace cho chain/node (P2P/PoW/RPC/DB/Types/Crypto).
- EGG-Kernel: phần kernel (chưa triển khai ở bước này).
- EGG-OS: phần hệ điều hành (chưa triển khai ở bước này).
- EGG-APP: phần ứng dụng (chưa triển khai ở bước này).
